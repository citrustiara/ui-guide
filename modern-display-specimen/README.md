# Modern Display Specimen UI Guide Pack

This folder contains a modern display-font specimen style package inspired by hard-edged typography posters, but recolored away from acid green into oxidized rust, pearl, black ink, and muted silver.

## Files

- `styleguide.md` - detailed UI and presentation guidelines.
- `tokens.css` - shared colors, typography, grid rules, and utility classes.
- `home.html` - short editable portfolio homepage example.
- `example-deck.html` - two-slide editable HTML deck in the same style.

## Preview

Open these files directly in a browser:

- `home.html`
- `example-deck.html`

For the deck, use left and right arrow keys to move between slides. Add `?qa=1` to hide the navigation bar for screenshots.
