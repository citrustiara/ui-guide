# Vibecoded Glassmorphism Baseline

This folder is a comparison baseline, not a recommended UI Guide style pack.

It intentionally uses the familiar modern AI-generated glassmorphism look: translucent cards, glow-heavy gradients, vague SaaS copy, oversized metrics, and soft dashboard panels. It does not import or reuse any `tokens.css` file or visual rule from the curated folders.

## Files

- `home.html` - standalone glassmorphism homepage example.
- `example-deck.html` - standalone two-slide glassmorphism deck example.
- `preview/` - screenshots used by the root showcase and comparison docs.

## Preview

Open these files directly in a browser:

- `home.html`
- `example-deck.html`

For the deck, use left and right arrow keys to move between slides. Add `?qa=1` to hide the navigation bar for screenshots. Add `?qa=1&slide=2` to inspect the second slide directly.
