# Futura Bauhaus UI Guide Pack

This folder contains a bold geometric Bauhaus-inspired style package based on the Futura showcase aesthetic.

## Files

- `styleguide.md` - detailed UI and presentation guidelines.
- `tokens.css` - shared colors, typography, layout tokens, and utility classes.
- `example-deck.html` - two-slide editable HTML deck in the style.
- `home.html` - short portfolio homepage example with placeholder content.

## Preview

Open these files directly in a browser:

- `example-deck.html`
- `home.html`

For the deck, use left and right arrow keys to move between slides. Add `?qa=1` to hide the navigation bar for screenshots.
